# Where To Go — Куда пойти?

Web-приложение для случайного выбора места на карте города.

## Что делает приложение

1. Показывает интерактивную карту города
2. Позволяет выбрать зону (полигон) — вручную или из пресетов
3. Пользователь(и) ставят свою позицию на карте (поддержка нескольких друзей)
4. Настраиваются мин/макс расстояния
5. Приложение генерирует случайную точку в зоне с учётом ограничений
6. Опционально: показывает маршрут до точки (пешком/транспорт)

## Стек технологий

- **Frontend**: Svelte 5 + Vite
- **Карта**: MapLibre GL JS + MapTiler (бесплатный тариф, красивые тайлы)
- **Гео-утилиты**: @turf/turf (point-in-polygon, haversine, random points)
- **Стили**: Tailwind CSS v4
- **Роутинг (v2)**: Valhalla public API
- **Backend**: нет, чисто клиентское приложение

## Дизайн

- Map-first layout: карта на весь экран, sidebar слева (desktop) / bottom sheet (mobile)
- Цвета: тёплые нейтральные (#F8F5F1 bg, #1A1A1A text) + teal акцент (#2B7A6F)
- Шрифты: Inter (body), Plus Jakarta Sans (headings)
- НЕ фиолетовый AI-стиль, а стильный минималистичный дизайн

## Команды

```bash
npm run dev    # запустить dev server
npm run build  # собрать для продакшена
npm run preview # превью билда
```
